This is the documentation for the **rudi command line utility**.
It will help you execute all RuDI-related tasks on a Linux server, 
including installation, management, and pipeline execution.

### Other ways to call RuDI functions

The RuDI command line utility (CLI) is only
used on a Linux server that will run HPC data analysis pipelines.
Here are some of the other ways RuDI tools are accessed:

- RuDI Desktop (recommended): <https://rustydataint.github.io/rudi-desktop-app>
- RuDI public web servers: <https://github.com/RustyDataInt/rudi-web-server>

In remote modes, the recommended RuDI Desktop will install
and makes calls to the CLI for you, via SSH.
